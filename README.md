# Lincoln Bohn

Mixed-methods research associate at **UCLA Integrated Substance Use and Addiction Programs (ISAP)**. Most of my work involves helping turn survey and interview data into something usable for decisions.

### What I work on
- **Survey research**: survey design, Qualtrics programming, field deployment, data extraction from paper forms, analysis, reporting.
- **Mixed methods**: staff and exit interviews, focus groups, fieldwork.
- **Applied data work**: Python (pandas, scikit-learn, some FastAPI), SQL (Postgres/SQLite), and some experimentation with VLM/Teleform pipelines and RAG.
- **Tooling**: contribute to public-facing platforms, dashboards, internal tools, and some mobile app work (in progress).

### A few projects
- **[super.ucla.edu](https://super.ucla.edu)** — Substance Use Evidence-Based Resource repository used by behavioral-health, educational, and service providers in California. I contributed to the design, build, and ongoing maintenance.
- Various internal tools (private): small full-stack apps, research databases, workflow automation for survey and interview data cleaning, and a home-server setup running some containerized services.
